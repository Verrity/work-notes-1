
# HK Code Block
* Add title

reset
* `collapse`. По умолчанию выключен - включить `unfold`
* `title`.   Заголовок задается как `title:"<title>"`

Примеры:
```c title:"Заголовок блока кода" unfold
printf("Hello world\n");
// Комментарий
printf("Hello world\n");
printf("Hello world\n");
printf("Hello world\n");
printf("Hello world\n");
printf("Hello world\n");
```
