### Регистры общего назначения ARM32

| Регистр      | Назначение                           |
| ------------ | ------------------------------------ |
| **R0**       | Первый аргумент функции              |
| **R1**       | Второй аргумент                      |
| **R2**       | Третий аргумент                      |
| **R3**       | Четвёртый аргумент                   |
| **R4**       | Сохраняемый (callee‑saved)           |
| **R5**       | Сохраняемый                          |
| **R6**       | Сохраняемый                          |
| **R7**       | Сохраняемый                          |
| **R8**       | Сохраняемый                          |
| **R9**       | Сохраняемый                          |
| **SL (R10)** | Stack Limit (опционально)            |
| **FP (R11)** | Frame Pointer                        |
| **IP (R12)** | Внутрипроцедурный (scratch)          |
| **SP (R13)** | Stack Pointer (указатель стека)      |
| **LR (R14)** | Link Register (адрес возврата)       |
| **PC (R15)** | Program Counter (текущая инструкция) |
